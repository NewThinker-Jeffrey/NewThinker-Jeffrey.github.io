数值稳定性：矩阵的条件数

条件数简介

求解线性方程组时，我们通常会谈及到矩阵的"条件数"（condition number）的概念。宽泛地讲，条件数用于衡量一个计算任务（可以抽象为一个函数）的输出对输入扰动的敏感程度，这个计算任务甚至可以是非线性的。
事实上，"矩阵AAA的条件数" 本身并不是一个足够清晰的概念：面对不同的计算任务时，AAA可以定义不同的条件数；比如，一个矩阵AAA，对于求特征值和求逆矩阵这两个不同的任务，它可以有不同的条件数，且即便它对求解特征值有很好的条件数，它对求逆运算的条件数也可能很差（A matrix can be poorly conditioned for inversion while the eigenvalue problem is well conditioned ）；
只不过，多数情况下，我们提到"矩阵AAA的条件数"时，暗指的是在求解 Ax=bAx=bAx=b 这样的线性方程组时的条件数。同样的条件数适用于对矩阵AAA的求逆运算。

线性方程组求解问题中的条件数

前面提到，"条件数" 用于衡量一个计算任务的输出对输入扰动的敏感程度。
对于求解线性方程组 Ax=bAx=bAx=b 这样一个计算任务，给定系数矩阵AAA后，它的输入为bbb，输出为xxx；
因此，我们可以定义矩阵 AAA （在解方程组这个任务中）的条件数κ(A)\kappa (A)κ(A)为： 解算结果 xxx 的相对误差 ∣δx∣∣x∣\frac{|\delta x|}{|x|}∣x∣∣δx∣​ 与 bbb 的相对误差 ∣δb∣∣b∣\frac{|\delta b|}{|b|}∣b∣∣δb∣​ 之比的上限，即
κ(A)=max⁡(∣δx∣∣x∣/∣δb∣∣b∣)(1)\kappa (A)=\max \left(\frac{|\delta x|}{|x|}/\frac{|\delta b|}{|b|}\right)\tag{1}κ(A)=max(∣x∣∣δx∣​/∣b∣∣δb∣​)(1)
注意，这里涉及到了 δb,b,δx,x\delta b,b, \delta x, xδb,b,δx,x 这四个向量的范数(∣δb∣,∣b∣,∣δx∣,∣x∣|\delta b|,|b|, |\delta x|, |x|∣δb∣,∣b∣,∣δx∣,∣x∣)。可见，即便对于解方程这个特定的计算任务，AAA的条件数的定义也不唯一：它取决于我们选择什么样的向量范数。
下面，我们详细展开这一常见的矩阵条件数的定义。
AAA 做一个非退化的矩阵（对应一个非退化的线性变换），它的诱导范数定义为

矩阵条件数的具体形式

如果向量范数选择欧氏范数（2-范数），容易证明，
κ(A)=∥A∥∥A−1∥=σmax(A)σmin(A)\kappa (A)= \Vert A \Vert \Vert A^{-1} \Vert = \frac{\sigma_{max}(A)}{\sigma_{min}(A)}κ(A)=∥A∥∥A−1∥=σmin​(A)σmax​(A)​
其中，σmax(A),σmin(A){\sigma_{max}(A)},{\sigma_{min}(A)}σmax​(A),σmin​(A) 分别矩阵 AAA 的最大、最小奇异值。实际工程中，尤其在AAA是维数很高的矩阵时，精确计算2-范数下的条件数几乎不可行（因为要计算奇异值），只能对其做一些量级估计。

证明：
考虑AAA的SVD分解 A=UDVTA=UDV^TA=UDVT，在2-范数下，我们有 ∣Ax∣=∣UDVTx∣=∣Dx∣|Ax|=|UDV^Tx|=|Dx|∣Ax∣=∣UDVTx∣=∣Dx∣ （这里用到了U,VU,VU,V都是正交阵的特性，正交阵作用于向量时不改变向量在2-范数下的长度）。而
∣Dx∣∣x∣=∑(σixi)2∑xi2≤∑(σmaxxi)2∑xi2=σmax\frac {|Dx|}{|x|}=\sqrt{\frac{\sum (\sigma_ix_i)^2} {\sum x_i^2}} \le \sqrt{\frac{\sum (\sigma_{max} x_i)^2} {\sum x_i^2}} = \sigma_{max}∣x∣∣Dx∣​=∑xi2​∑(σi​xi​)2​​≤∑xi2​∑(σmax​xi​)2​​=σmax​

同理，∥A−1∥=1σmin\Vert A^{-1} \Vert=\frac{1}{\sigma_{min}}∥A−1∥=σmin​1​（注意到A−1A^{-1}A−1的最大奇异值是AAA的最小奇异值之倒数）。
因此有 κ(A)=∥A∥∥A−1∥=σmax(A)σmin(A)\kappa (A)= \Vert A \Vert \Vert A^{-1} \Vert = \frac{\sigma_{max}(A)}{\sigma_{min}(A)}κ(A)=∥A∥∥A−1∥=σmin​(A)σmax​(A)​

重要特例：

  • 如果AAA是 normal matrix (比如实对称矩阵)，κ(A)\kappa (A)κ(A) 等于AAA的模长最大、最小特征值的模长之比；
  • 如果AAA是酉矩阵，比如单位阵，则κ(A)=1\kappa (A) = 1κ(A)=1

LU 分解法

LU 分解法求解线性方程组本质上就是高斯消元法；

假设 AAA 是满秩的方阵，我们用 A=LUA=LUA=LU 代表 AAA 的 LU 分解，其中L,RL,RL,R分别是下三角矩阵和上三角矩阵。
LU分解本质是对矩阵 AAA 做基础行变换进行高斯消元，把A变成上三角阵。所有基础行变换的作用组合在一起，等价于左乘一个下三角矩阵L−1L^{-1}L−1 (注意下三角阵的逆依然是下三角)，消元后剩下的上三角矩阵就是RRR。
对线性方程组右侧的bbb做相同的基础行变换后得到b′=L−1bb'=L^{-1}bb′=L−1b，然后 back substitution回代求解 Rx=b′Rx=b'Rx=b′即可得到原问题的解。

关于LU分解的唯一性：

  • 不对LU做额外要求的话，LU分解不唯一（比如对高斯消元后的上三角矩阵R的每行再乘一个非零系数，就可以得到一个不同的上三角阵R'）；
  • 对分解做某些特定限制后，比如要求LLL的对角线都是1、或者RRR的对角线都是1等等，LU分解才唯一；

思路简介

共轭梯度法求解 Ax=bAx=bAx=b 或 Ax−b=0Ax-b=0Ax−b=0
使用条件：AAA是对称正定阵
考虑二次函数 f(x)=12xTAx−bTxf(x)=\frac{1}{2}x^TAx - b^Txf(x)=21​xTAx−bTx，其一阶导为f′(x)=Ax−bf'(x)=Ax-bf′(x)=Ax−b。
求解 Ax−b=0=f′(x)Ax-b=0=f'(x)Ax−b=0=f′(x) 等价于求解 arg⁡min⁡xf(x)\arg \min_x f(x)argminx​f(x) 。这是一个二次优化问题，可以用优化的方法来求解。

为理解共轭梯度法，首先要理解共轭向量的概念：
共轭向量：向量pip_ipi​与pjp_jpj​是共轭的，如果piTApj=0p_i^TAp_j=0piT​Apj​=0；

  • 一组两两共轭的向量{pi}\{p_i\}{pi​}，必然是线性无关的，否则与AAA的正定性矛盾。比如，若αp1+βp2=p3\alpha p_1+ \beta p_2= p_3αp1​+βp2​=p3​，那么由AAA的正定性有p3TAp3>0p_3^TAp_3>0p3T​Ap3​>0，但另一方面由{pi}\{p_i\}{pi​}两两之间的共轭性，有 p3TAp3=αp1TAp3+βp2TAp3=0p_3^TAp_3 = \alpha p_1^TAp_3 + \beta p_2^TAp_3 = 0p3T​Ap3​=αp1T​Ap3​+βp2T​Ap3​=0，矛盾；
  • 一组两两共轭的向量{pi}\{p_i\}{pi​}，最多包含nnn个向量（因为它们必须线性无关）；如果{pi}\{p_i\}{pi​}包含nnn个共轭向量则称其为一个极大共轭向量组；

由于AAA的对称和正定性，二次函数f(x)f(x)f(x)的"等高面"是"椭球面"；椭球的球心即待求的解；
任意给定初始点x0x_0x0​，该点的梯度方向 (垂直于所在椭球面) 为 f′(x0)=Ax0−bf'(x_0)=Ax_0-bf′(x0​)=Ax0​−b，hessian阵为 AAA；
从x0x_0x0​出发，为了到达球心（球心梯度为0），就要找一个前进方向抵消掉x0x_0x0​处的梯度f′(x0)f'(x_0)f′(x0​)，因此x0x_0x0​处的最佳前进向量 ppp 要满足 Ap=−f′(x0)=−(Ax0−b)Ap=-f'(x_0)=-(Ax_0-b)Ap=−f′(x0​)=−(Ax0​−b)，即 p=A−1b−x0p=A^{-1}b-x_0p=A−1b−x0​；
可见想直接得到最佳前进向量 ppp，需要计算A−1A^{-1}A−1，而这正是我们想要避免的操作；
那我们就要想办法，能否"间接"地到达球心？
共轭方向法提供了一种思路：给定一个极大共轭向量组{pi}\{p_i\}{pi​}，从x0x_0x0​出发，依次沿各个共轭方向走到该方向的极小点，注意这个极小点相切于某个等高椭球面。这样，最多n步，就能到达球心（如果通过一个线性变换把解空间映射为一个新的线性空间，在新的线性空间上所有等高面变为球面，那这组共轭向量就变成了新空间的一组正交基或坐标轴，依次沿这些向量找切点相当于依次沿这组坐标轴走到原点，因为只有n个轴，所以最多走n步就能到原点）。

对于二次函数f(x)f(x)f(x)，沿各方向找极小点（切点）可转化为找一个一元二次函数的极小点，可直接解析地算出该点，不需要 line search和迭代。

共轭梯度法用于求解一般的最优化问题时（非线性），每一步的前进步长 αk\alpha_kαk​ 是需要 line search 的。实际中可选 exact line search 或 soft line search；

共轭梯度法是一种具体的共轭方向法。依据初始点 x0=0x_0=0x0​=0 的梯度，以及后续各切点xkx_kxk​ 的梯度和前一步的前进方向的线性组合，来确定下一步的前进方向，使得第k+1k+1k+1步的前进方向与前kkk步的前进方向保持共轭。
具体公式不再写出。见 Chapter_15__Conjugate_Gradient 15.2节，附件有备份；

共轭梯度法的好处在于: 每步迭代的最大的计算量仅来源于矩阵向量乘法, 并且不需要对系数矩阵进行修改（做矩阵分解或消元）， 因此不会引入 fill-in 的问题.

收敛速度，矩阵AAA的条件数的影响

这里主要从几何角度做定性分析。
理论上，对于nnn维的解空间，用共轭梯度法求解二次函数的最小值，最多需要迭代nnn步。对于nnn特别大的情况，我们不希望算法走完所有这nnn步。
我们知道，共轭梯度法在初始点x0=0x_0=0x0​=0处的搜索方向p0p_0p0​是其负梯度方向p0=−f′(x0)p_0=-f'(x_0)p0​=−f′(x0​)；xkx_kxk​ 是子空间 span(f′(x0),f′(x1)...f′(xk−1))span(f'(x_0), f'(x_1) ... f'(x_{k-1}))span(f′(x0​),f′(x1​)...f′(xk−1​)) 上的最小值点，在xkx_kxk​处的搜索方向pk∈span(f′(x0),f′(x1)...f′(xk))p_k\in span(f'(x_0), f'(x_1) ... f'(x_k))pk​∈span(f′(x0​),f′(x1​)...f′(xk​))；

  • 最理想的情况下，如果AAA是单位阵（这时 κ(A)=κ(I)=1\kappa(A) = \kappa(I) = 1κ(A)=κ(I)=1）， 二次函数 f(x)f(x)f(x) 的"等高面" 是 "球面"，x0x_0x0​的负梯度方向f′(x0)f'(x_0)f′(x0​)将直接指向"球心"ccc，因此只需1次迭代，我们就能到达f(x)f(x)f(x)的最小值点（即球心ccc）；
  • 如果AAA不是单位阵，但它的条件数比较小（AAA的各特征值相差不大），f(x)f(x)f(x) 的"等高面" 也是近似 "球面"，各点的负梯度方向也都是指向球心ccc附近，而每一步迭代的方向 pkp_kpk​ 也接近xkx_kxk​处的负梯度方向（指向球心附近），所以收敛就比较快，可能只需几步就能收敛到距离解很近的位置；
  • 而反过来，如果AAA的条件数很大，那 f(x)f(x)f(x) 的"等高面" 就是非常椭的 "椭球面"（想象一个飞盘的表面，并极限压扁），这个"椭球面"上大部分点的负梯度方向（垂直于椭球面的方向），与该点到球心的连线方向夹角都很大，甚至可能是接近垂直的！这种情况下迭代过程中，xkx_kxk​刚开始会来回小幅度震荡（接近静止不动），且 pkp_kpk​ 与 xkx_kxk​到球心的方向(c−xk)(c-x_k)(c−xk​) 接近正交 （因为 pk∈span(f′(x0),f′(x1)...f′(xk))p_k\in span(f'(x_0), f'(x_1) ... f'(x_k))pk​∈span(f′(x0​),f′(x1​)...f′(xk​))，而f′(xi)f'(x_i)f′(xi​)都与(c−xk)(c-x_k)(c−xk​)接近正交），靠近球心的速度非常缓慢，导致收敛极慢。

下式定量地描述了收敛速率的下限：
∥xk−x∗∥A≤2∥x0−x∗∥A(κ(a)−1κ(a)+1)k\left\| x_k - x^* \right\|_A \leq 2\left\| x_0 - x^* \right\|_A(\frac{\sqrt{\kappa (a)} - 1}{\sqrt{\kappa (a)} + 1})^{k}∥xk​−x∗∥A​≤2∥x0​−x∗∥A​(κ(a)​+1κ(a)​−1​)k
κ(a)\kappa (a)κ(a) 是椭球面的最长轴与最短轴之比；
可见共轭梯度法保证至少 (κ(a)−1κ(a)+1)(\frac{\sqrt{\kappa (a)} - 1}{\sqrt{\kappa (a)} + 1})(κ(a)​+1κ(a)​−1​) 倍的线性收敛速率。

PCG: Preconditioned CG

改善条件数可以加快收敛速度。
precondition的操作，是选择一个合适的矩阵MMM，把方程组Ax=bAx=bAx=b 变成 M−1Ax=M−1bM^{-1}Ax=M^{-1}bM−1Ax=M−1b，且M−1AM^{-1}AM−1A 有比较好的条件数；
一个最简单的 preconditioner 是 diagonal 或 Jacobi preconditioner。即选取AAA的diagonal或block diagonal作为MMM。
这个preconditioner不用求大矩阵的逆，但依然要用到 "实数的逆(即倒数)"或"小矩阵块的逆"。各个小矩阵块自身的条件数一般远小于整个大矩阵的条件数， 对小矩阵块单独求逆一般在数值上也是相对稳定的。
另外，PCG这里加 preconditioner 的主要目的，不是为了增加数值稳定性，而是为了增加收敛效率。
