General MLE

Let's define some symbols first to model the problem:

  • SSS: the state space. a manifold.

  • M≡RmM\equiv \mathbb R^mM≡Rm: the measurement space (observation space). an Euclidean space.

  • s∗∈Ss^*\in Ss∗∈S: the real (unobservable) state. it should be considered as an unknown constant, rather than a random point.

  • s∈Ss \in Ss∈S: the state variable, an arbitrary guess for s∗s^*s∗

  • f:S→Mf: S\rightarrow Mf:S→M: the observation function.

  • nyn_yny​: the observation noise, a random vector in Rm\mathbb R^mRm, ny∝N(0,Σ)n_y\propto \mathcal N (0, \Sigma)ny​∝N(0,Σ) , for simplicity we choose Σ=I\Sigma=IΣ=I.

  • yyy: the observation (before the experiment), a random vector in Rm\mathbb R^mRm depending on nyn_yny​: y=f(s∗)+nyy=f(s^*) + n_yy=f(s∗)+ny​. y∝N[f(s∗),Σ=I]y\propto \mathcal N [f(s^*) , \Sigma=I]y∝N[f(s∗),Σ=I]

  • y∗y^*y∗: the realized observation, a constant, y∗∈M≡Rmy^* \in M\equiv \mathbb R^my∗∈M≡Rm,

  • eS:S→Rme_S: S\rightarrow \mathbb R^meS​:S→Rm: the error function, a short hand for f()−y∗f()-y^*f()−y∗. To be more specific, eS(s)=f(s)−y∗e_S(s) = f(s) - y^*eS​(s)=f(s)−y∗

According to the model, the probability (before the experiment) that yyy happens to be y∗y^*y∗ is:

p(y=y∗)=p[ny=y∗−f(x∗)=−eS(s∗)]=N[−eS(s∗)]∝exp⁡[−12eS(s∗)TeS(s∗)]p(y=y^*) = p[n_y=y^*-f(x^*)=-e_S(s^*)] = \mathcal N[-e_S(s^*)]\propto \exp[-\frac{1}{2}e_S(s^*) ^Te_S(s^*) ]p(y=y∗)=p[ny​=y∗−f(x∗)=−eS​(s∗)]=N[−eS​(s∗)]∝exp[−21​eS​(s∗)TeS​(s∗)]

the gradient and hession of F(x)F(x)F(x) is:

An interesting viewpoint

The whole process of realizing an observation y∗y^*y∗ and then doing MLE (for x∗x^*x∗) with it can be considered as getting a sample for the random vector xEx^ExE, and xE∗x^{E*}xE∗ is the sample. If we use this sample to estimate (also by MLE) xEx^ExE's expectation μ=x∗\mu=x^*μ=x∗, then we get xE∗x^{E*}xE∗ again.

If the state space SSS is Euclidean and the observation function is strictly affine (so we can set x0=0x_0=0x0​=0 without loss of generality, and Tx0S=ST_{x_0}S=STx0​​S=S which implies ⊞\boxplus⊞ is simply "+++", s=x+s0=xs=x+s_0=xs=x+s0​=x, s∗=x∗+s0=x∗s^*=x^*+s_0=x^*s∗=x∗+s0​=x∗), and the noise nyn_yny​ is Gaussian, then the following two estimation problems are equivalent:

  1. Given the observation variable y=f(s∗)+ny=f(x∗)+nyy=f(s^*)+n_y=f(x^*)+n_yy=f(s∗)+ny​=f(x∗)+ny​ and a realized sample y∗y^*y∗; Then we use the sample y∗y^*y∗ to estimate (by MLE) x∗x^*x∗;

  2. The random vector xEx^ExE can be viewed as a pseudo-observation, and xE∗x^{E*}xE∗ is a sample or realization of xEx^ExE; Then we use the sample xE∗x^{E*}xE∗ to estimate (also by MLE) xEx^ExE's expectation μ=x∗\mu=x^*μ=x∗

The likelyhood function of 1 is: L1(x)=p(y=y∗)∣if x∗=x∝exp⁡{−12[(x−μy∗)TH(x−μy∗)]}L_1(x)=p(y=y^*)|_{\text {if } x^*=x}\propto \exp\{-\frac{1}{2}[(x-\mu_{y^*})^TH(x-\mu_{y^*})]\}L1​(x)=p(y=y∗)∣if x∗=x​∝exp{−21​[(x−μy∗​)TH(x−μy∗​)]};

The likelyhood function of 2 is: L2(x)=p(xE=xE∗)∣if μ=x∝exp⁡{−12[(xE∗−x)TH(xE∗−x)]}L_2(x)=p(x^E=x^{E*})|_{\text {if } \mu=x}\propto \exp\{-\frac{1}{2}[(x^{E*}-x)^TH(x^{E*}-x)]\}L2​(x)=p(xE=xE∗)∣if μ=x​∝exp{−21​[(xE∗−x)TH(xE∗−x)]};
L1(x)=L2(x)L_1(x)=L_2(x)L1​(x)=L2​(x) since μy∗=xE∗\mu_{y^*}=x^{E*}μy∗​=xE∗, i.e. the two problem give the save likelyhood function.

If y∗y^*y∗ is just part of the observation of another larger problem, then we can remove this part from the larger problem and insert back the pseudo-observation part xE∗x^{E*}xE∗. Note xE∗x^{E*}xE∗ should be the MLE result for the the real states by only considering the y∗y^*y∗ part of the total observation; Bacause y∗y^*y∗ and xE∗x^{E*}xE∗ will give the same partial-likelyhood.

通常在工程上，边缘化的作用在于从滑窗中移除一些时间上较老的状态量和与之关联的观测量，以确保计算量有界，而又尽可能少地损失信息（之前已说明，如果误差函数是严格的仿射函数且噪声都是高斯的，则不损失信息，这时可以理想地进行"增量式"MLE或LSE）。
比如，假设aaa对应时间比较老的状态量，我们准备将这部分状态量以及相关的观测量y∗y^*y∗移除掉。
首先再梳理一下符号：

  • xEx^ExE: 在获知xE∗x^{E*}xE∗前对xE∗x^{E*}xE∗的推测，这是一个随机变量；反过来看，xE∗x^{E*}xE∗可看做是对随机变量xEx^ExE的一次采样；

  • t时刻，我们首先用上一节的舒尔补消元方法将ttt时刻的b∗b^*b∗计算出来；a∗a^*a∗ 我们不做计算，而且之后ttt时刻之后也不再需要用它的值；
